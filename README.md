# Student-Record-Saver
A Python project that stores a student details ina text file and reads them back.
## Features
Takes student details as input
Stores data in a text file
Reads data from the file
Uses file handling concepts
Demonstrates data persistence
## Inputs
-NAME
-AGE
-BRANCH
-CGPA
## Example Output
NAME: Nithya
AGE: 18
BRANCH: AIML
CGPA: 9.27
## Concepts Used
-Variables
-Input/Output
-Lists
-Loops
-File Handling
-Read and Write Operations
